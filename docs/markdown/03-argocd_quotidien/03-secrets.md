## Gérer des secrets dans Git

- Utiliser un gestionnaire de secret externe (Vault, etc...) <!-- .element: class="fragment fade-in-then-semi-out"-->

- Bitnami SealedSecrets, SOPS <!-- .element: class="fragment fade-in-then-semi-out" -->
